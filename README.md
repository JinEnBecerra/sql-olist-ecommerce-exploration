# Olist E-commerce Data Analysis  

This project analyzes a Brazilian e-commerce dataset (Olist) to understand business performance, customer experience, and key revenue drivers. It is designed to simulate a real-world business analysis scenario and demonstrate how data can support decision-making.

The goal is to answer practical business questions using SQL and present the results through interactive Tableau dashboards.

---

## Project Overview  

This analysis focuses on three main areas:

- Business performance over time  
- Delivery and customer experience  
- Key factors driving revenue  

The dataset covers transactions from 2016 to 2018.

---

## Tools Used  

- **SQL (PostgreSQL)** – data cleaning, transformation, and analysis  
- **Tableau** – dashboard design and visualization  
- **GitHub** – project documentation

## Key Insights  

- Revenue increased steadily over time, with a peak in late 2017  
- Growth was mainly driven by more orders, not higher spending per order  
- Delivery performance improved after the early stage of the platform  
- Customer satisfaction remained consistently high  
- Revenue is concentrated among a small number of states, categories, and sellers  
- Some high-selling product categories have relatively low ratings, which may impact long-term customer satisfaction
